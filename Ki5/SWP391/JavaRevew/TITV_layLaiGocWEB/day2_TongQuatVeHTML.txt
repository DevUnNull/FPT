III. name , type , id , value , for 

<input type="radio" name="cpu" id="i7" value="Core i7">
<label for="i7">Core i7</label>

1. name : name="cpu" tên biến mà jsp nhận 
	- Khi bạn dùng request.getParameter("cpu") trong JSP, bạn đang gọi tới name="cpu".

2. value :value="Core i7" → Giá trị gửi về server
	- Đây là giá trị thực sự được gửi lên khi bạn chọn radio hoặc checkbox.
	- Bạn có thể hiển thị, lưu database, tính tiền,… dựa vào value này.
3. id : id="i7" → Mã định danh cho DOM + liên kết với label
	- Đây là tên duy nhất của input trong trang.
	- Không ảnh hưởng đến server, nhưng rất quan trọng để:	
	- liên kết với <label>
	- điều khiển bằng JavaScript
4. for : for="i7" (trong label) → Liên kết label và input
	- Cho phép click vào chữ “Core i7” để chọn luôn radio.
	- Dùng for="id của input".

=> tổng kết 
	- name dùng để gọi trong jsp
	- value dùng để định giá tiền hoặc cái gì đó , khi mình submit thì radio được chọn đó sẽ gửi giá trị đó về để mình dùng
	- id là để định danh cái radio đó
	- for là để liên kết với radio mà mình muốn , nếu như ấn vào chữ đã liên kết với for thì nó sẽ gọi đến radio mà for liên kết bằng id
